- ## Project Overview

This project is an audit of **Git** as a free and open‑source software (FOSS) tool.  
It has two main parts:

1. A written report (PDF) covering:
   - Origin and philosophy of Git  
   - License analysis (GPLv2)  
   - Ethics of open source  
   - Linux footprint  
   - FOSS ecosystem around Git  
   - Comparison of Git with a proprietary alternative (Perforce Helix Core)

2. Five Bash shell scripts that demonstrate practical Linux and shell‑scripting concepts using Git as the example software. [file:1]
